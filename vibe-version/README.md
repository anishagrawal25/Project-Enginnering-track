# Vibe Version

This folder contains the task manager app built using a vibe coding tool.

**Tool used:** v0.dev  
**Time to generate:** 8 minutes
**Prompt used:**
Build a simple React task manager app with:

- Add task
- Mark task complete
- Filter: All / Active / Completed
- Clean UI with dark mode
  **Live URL:** https://anishagrawal25.github.io/Project-Enginnering-track/vibe-version/

## Notes

The tool generated a complete task manager application very quickly with a clean UI and dark/light mode support. It automatically created multiple components and a structured folder system.

However, the generated code was slightly complex and harder to understand. Some logic was distributed across different files, which made it difficult to trace and modify.

Overall, the tool was very fast for building a working app, but it provided less control and required extra effort to understand the code.
