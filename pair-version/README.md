# Pair Version

This folder contains the task manager app built using an AI pair programming assistant.

**Tool used:** GitHub Copilot  
**Time to build:** ~45 minutes

**Suggestions accepted:** ~15  
**Suggestions rejected:** ~5

**Live URL:** https://anishagrawal25.github.io/Project-Enginnering-track/

## Notes

In this version, the app was built step-by-step with the help of GitHub Copilot, acting like a pair programming partner.

Unlike the vibe version, the code was written with full understanding. Each feature such as adding tasks, marking them complete, and deleting tasks was implemented manually with guidance from AI suggestions.

Copilot helped by suggesting code snippets, especially for functions and DOM manipulation, which improved speed. However, some suggestions were not fully correct or suitable, so they were modified or rejected.

This approach provided better control over the code and improved understanding of how the application works.

Overall, this method was slower than vibe coding but much better for learning and explaining the logic.
